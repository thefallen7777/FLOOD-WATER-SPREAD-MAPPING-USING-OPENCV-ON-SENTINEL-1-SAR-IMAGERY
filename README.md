# FLOOD-WATER-SPREAD-MAPPING-USING-OPENCV-ON-SENTINEL-1-SAR-IMAGERY
OpenCV-based flood mapping using Sentinel-1 SAR imagery from the Sen1Floods11 dataset. The system segments water regions via intensity thresholding and morphological refinement, evaluates performance using IoU and F1-score, and visualizes predicted flood areas for rapid disaster assessment and construction risk analysis.
